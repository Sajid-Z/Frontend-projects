# Password Generator App

A simple and efficient password generator application that allows users to create strong, randomized passwords instantly.

## Features

* Generate **two random passwords** at once
* Specify custom **password length**
* One-click **generate button**
* **Copy to clipboard** functionality for convenience
* Fast and lightweight

## Work flow
 Enter your desired password length. Click the **Generate** button. After that, The app will instantly generate two secure random passwords. Then Click the Copy button to copy your preferred password to the clipboard.
