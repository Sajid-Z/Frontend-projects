# Password Generator App

A simple and efficient password generator application that allows users to create strong, randomized passwords instantly.


## Features

* Generate **two random passwords** at once
* Specify custom **password length**
* One-click **generate button**
* **Copy to clipboard** functionality for convenience
* Fast and lightweight

## How It Works

1. Enter your desired password length.
2. Click the **Generate** button.
3. The app will instantly generate **two secure random passwords**.
4. Click the **Copy** button to copy your preferred password to the clipboard.

## Tech Stack

* HTML
* CSS
* JavaScript

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/password-generator.git
   ```

2. Navigate to the project folder:

   ```bash
   cd password-generator
   ```

3. Open `index.html` in your browser.
